# DPDP Readiness Checklist

Use this before onboarding pilot firms.

- [ ] Confirm what personal data is collected for users: name, email, role, organization, deal assignments, reminder preferences.
- [ ] Publish user-facing notice covering purpose, retention, subprocessors, and support contact.
- [ ] Add consent or contractual basis capture for account creation and reminder emails.
- [ ] Add per-user reminder subscription preferences and unsubscribe handling.
- [ ] Prepare processor agreement language for law-firm customers.
- [ ] Define retention windows for closed deals, deleted organizations, notes, audit entries, and email logs.
- [ ] Define breach notification workflow and owner.
- [ ] Maintain subprocessors list for Vercel, Neon, Clerk, Resend, Inngest, Sentry, and analytics providers.
- [ ] Verify that exports do not include privileged document contents.
- [ ] Verify that notes and comments remain length-capped and status-only.
